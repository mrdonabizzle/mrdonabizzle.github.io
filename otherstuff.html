Raspberry Pi

I have been interested in Raspberry Pi for a few years now. I have built several RetroPie gaming systems and used a few of them with Raspbian as a cheap alternative desktop. Last year, I incorporated several others into robotics projects using using the GoPiGo kits from Dexter Industries. Currently I am building a PiGrrl, which is a handheld that looks like a Game Boy.
